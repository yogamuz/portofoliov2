Arsitektur sistem Tabungan Digital IoT
Alur data dari hardware ESP32 menuju backend Express, lalu ditampilkan realtime di frontend React
Hardware
ESP32
terhubung WiFi 2.4GHz
Sensor TCS34725
membaca warna uang, hitung rasio R/G dan B/G
Klasifikasi nominal
cocokkan rasio warna ke threshold Rp10K - Rp100K
Keypad 4x4
input PIN 6 digit, auto-submit dan masking di LCD
LCD 16x2 I2C
tampilkan status scan, PIN, dan hasil transaksi
Firmware ESP32
state machine MODE_SCAN / MODE_KEYPAD, kirim HTTP POST
HTTP POST
JSON + Bearer token
Backend
Express API
Node.js, port 3000
Middleware auth
validasi header Authorization Bearer ESP32_SECRET_TOKEN
Route transaksi
simpan transaksi baru, hitung dan update saldo
Firebase Firestore
collections TRANSAKSI dan SETTINGS
SSE broadcast
endpoint /api/transaksi/stream, push event ke semua client
Server-Sent
Events (push)
Frontend
React app
localhost:5173, Neobrutalism UI
Hook useTabungan
EventSource SSE, fetch summary dan transaksi otomatis
Dashboard
saldo, progress target, mapping warna uang
Transaksi
tabel riwayat dengan filter deposit / withdraw
Pengaturan
ubah target, ganti PIN, reset saldo
